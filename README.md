
## Quick builder

* Download https://www.raspberrypi.org/downloads/raspberry-pi-os/
* inspect/edit [wpa_supplicant.conf](wpa_supplicant.conf) in this folder.
* Run mkraspi.sh
* put SD card in Pi.
* ssh pi.local
